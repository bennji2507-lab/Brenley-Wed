## Upload to Netlify

1. Unzip this file.
2. Open Netlify.
3. Add new site -> Deploy manually.
4. Drag the unzipped folder named `brendan_linley_wedding_site_v3` into Netlify.
5. Do not upload the ZIP itself.
6. Once uploaded, open the Netlify link and enter password `belltower`.

## Important

The password gate is a simple front-end guest gate. It is suitable for a wedding site, but not for sensitive private information.
